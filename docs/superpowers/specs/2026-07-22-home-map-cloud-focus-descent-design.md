# 首页到地图：高空落地、云层与聚焦蒙版动效设计

## 目标

重做首页进入地图的第二阶段视觉合成，使地图从高空落下并停稳，地图顶部 UI 抢先出现，单份云层与白色蒙版同步飘入，最终将视觉焦点落在当前已解锁的阅读小屋。消除重复云层、三角裁切接缝、状态栏重影和导航后的二次加载感。

## 图层结构

转场舞台从下到上依次为：

1. `map-transition-base-lanhu.png`：无云地图底图，负责高空落地缩放。
2. 地图顶部 UI：复用 `map-final-lanhu.png` 的顶部区域，短时提前进场。
3. `map-house.png`：当前已解锁阅读小屋、角色和关卡标签，独立落位。
4. 白色聚焦蒙版：CSS 径向渐变，已解锁小屋附近保持清晰，其他区域逐渐被白雾覆盖。
5. `map-cloud-overlay.png`：只渲染一次，整体平移、缩放并落在最终位置，不再复制成四个三角裁片。
6. `map-final-lanhu.png`：最终完整地图，仅在停稳阶段无缝接管上述临时组合。
7. 素材加载失败时的白雾回退层。

## 时间轴

| 时间 | 阶段 | 行为 |
| --- | --- | --- |
| 0–150ms | 首页退场开始 | 卡片推开与 hero 渐隐继续使用现有节奏。 |
| 150–350ms | 顶部 UI 抢先进场 | 地图返回、标题、星星、成就和状态区短促下落并淡入。 |
| 260–1050ms | 地图高空落地 | 无云地图由 `scale(1.16)` 缩至 `scale(0.985)`，再回到 `scale(1)`；轻微模糊同步归零。 |
| 420–720ms | 已解锁关卡落位 | 阅读小屋、角色和标签短促落下并回弹。 |
| 520–1180ms | 云层与蒙版进入 | 单份云层和白色聚焦蒙版同步由屏幕下方向最终位置上浮并缩至 100%。 |
| 900–1250ms | 聚焦收拢 | 白雾完成覆盖，清晰窗口落在阅读小屋，其他关卡保持雾化。 |
| 1250–1500ms | 最终接管 | 完整地图渐显，临时底图、UI、小屋、蒙版和动态云层同步退出；最终画面不跳变。 |
| 1500ms | 页面导航 | 导航到带版本参数的 `level-map.html`，相同完整地图保持 100% 不透明且无二次动画。 |

## 动效参数

- 顶部 UI：200ms，`translateY(-12px) → 0`，使用快速 ease-out。
- 地图：790ms，`scale(1.16) → scale(0.985) → scale(1)`，`blur(3px) → 0`。
- 小屋：300ms，`translateY(-24px) scale(1.08) → translateY(0) scale(1)`。
- 云层与蒙版：660ms，`translateY(34%) scale(1.06) → translateY(0) scale(1)`。
- Reduced Motion：不执行位移、缩放、模糊或云层动画，仅在 180ms 内显示最终地图并导航。

## 交互与异常

- 保留双击防重、键盘 Enter、单次导航、storage handoff 与 bfcache 返回恢复。
- 素材预加载去重；任一关键素材失败时显示白雾回退层，但不阻塞导航。
- 地图页三个热点的位置和目标保持不变。
- 首页现有卡片推开和 hero 1000ms 渐隐节奏保持不变。

## 测试与验收

- TDD 更新转场时间常量与状态边界：地图 260ms、云层 520ms、接管 1250ms、导航 1500ms。
- 静态测试确保只有一份云层元素，不存在四向 `clip-path` 云片。
- 静态测试锁定顶部 UI、地图高空落地、小屋、蒙版、云层与最终地图的层级及动效参数。
- Reduced Motion 测试确保没有 transform、filter 和云层移动。
- 393×852 浏览器关键帧验证：顶部 UI 抢先出现、地图由大到小停稳、云层无接缝、白雾聚焦小屋、导航前后画面一致、控制台无错误。

## 范围外

- 不修改 `level-map.html` 的三个热点目标与位置。
- 不修改 Reading Hut 星星余额、兑换或放置逻辑。
- 不重新设计蓝湖地图素材本身。
